Display: Block vs Inline

Inline element (span) Another inline element (span)

Block elements start on a new line and take full width. Inline elements stay in the flow and only take as much width as needed.

Position: Relative, Absolute, Fixed

Relative (parent)
Absolute (child)
Fixed (always in top-right corner)

Relative: positioned relative to its normal place.
Absolute: positioned absolutely to the nearest positioned ancestor (here, the relative div).
Fixed: positioned relative to the viewport, stays visible when scrolling.
